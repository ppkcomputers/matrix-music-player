# mkv-stream OSD Music Player

Quickshell / Hyprland overlay music player with YouTube support.

### Features
- Global hotkey toggle (Super + M)
- Search & play YouTube videos directly
- Audio visualizer + Matrix rain background
- Real-time volume control
- Similar songs suggestions
- Lightweight mpv backend

Toggles on/off via `toggle-osd-nokill.sh`

Give it a keybinding in hyprland.conf
Example
bind = SUPER, M, exec, /home/user/.config/Quickshell/music-player/toggle-osd-nokill.sh
